Повторяющие вычисления - циклы.

Циклом в программировании называется многократное повторение некоторых операторов. Необходимость циклов несомненна. Например, нужно вычислить сумму целых чисел в некотором диапазоне, указанном пользователем, или найти факториал числа. Без циклов решение таких задач невозможно.

В JavaScript реализовано 3 вида циклов, рассматриваемых ниже.

  • Цикл со счетчиком "for"
    Этот тип цикла применяется в тех случаях, когда количество повторений известно заранее. Синтаксис конструкции "for" представлен ниже:
                for([инициализация_счетчика];[условие];[приращение_счетчика]) {
                оператор;
                оператор;
                ...
            }
    Примечание: в квадратных скобках "[]" указаны необязательные части конструкции.
    где:
    инициализация_счетчика - выражение, инициализирующее переменную - счетчик цикла.
    Может быть опущено, в случае, если переменная была проинициализированна до цикла.
    условие - условие на значение счетчика цикла. Пока условие истинно, цикл выполняется.
    приращение_счетчика - выражение, определяющее изменение (инкремент или декремент) счетчика на каждой итерации цикла.
  • Цикл с предусловием "while"
    Синтаксис:
            while(условие_продолжения) {
            оператор;
            оператор;
            ...
        }
    Принцип функционирования данной конструкции достаточно прост: сначала проверяется значение логического выражения условие_продолжения; если оно истино, выполняется тело цикла, иначе, управление передается первому оператору, следующему за циклом (после символа "}"). Именно поэтому цикл "while" и называют циклом с "предусловием".
    Будьте внимательны при построении условия продолжения цикла - если поставить заведомо неверное условие, тело цикла не выполнится ни разу!
    Цил "while" применяется чаще всего в тех случаях, когда точное количество повторений неизвестно. Однако, его можно применять так же, как цикл "for".

  • Цикл с постусловием "do ... while"
    Синтаксис: 
            do {
            оператор;
            оператор;
            ...
        }
        while(условие_продолжения)
    Принцип функционирования: на каждой итерации сначала выполняются операторы тела цикла, а, затем, проверяется значение выражения условие_продолжения. Если выражение имеет значение "истина", переходим к следующей итерации, иначе, управление передается первому оператору, следующему за циклом.
    Таким образом, цикл с постусловием выполняет тело цикла как минимум 1 раз.

Управление циклом

В JavaScript циклами можно управлять с помощью двух операторов:

  • break - прерывает выполнение цикла и передает управление на следующий после цикла оператор.
  • continue - прерывает текущую итерацию цикла и передает управление на первый оператор тела цикла для новой итерации.

Конструкция выбора "switch ... case ... default ..."

Оператор switch

Конструкция выбора используется в тех случаях, когда приходится делать выбор из множества альтернатив. Ниже представлен общий вид конструкции:

Синтаксис:

switch ( переменная ) {
    case  значение 1 :  оператор;  break;
    case  значение 2 :  оператор;  break;
    case  значение 3 :  оператор;  break;
    default :   оператор;
}

Принцип работы данной конструкции достаточно прост. Сначала вычисляется значение выражения ключевое_выражение. Затем проверяются последовательно блоки case значение_x:. Если значение выражения равно значению, заданному в текущем блоке case, выполняются операторы данного блока до оператора break. Если оператор прерывания цикла в текущем блоке не присутствует, выполняются операторы следующего за текущим блока case. Если значению ключевого выражения не соответствует ни один из перечисленных блоков case, выполняются операторы блока default (если он указан).

Существует ограничение, накладываемое на применение конструкции выбора: конструкция выбора применяется только в тех случаях, когда необходимо проверить значение ключевого выражения на равенство одному из фиксированных значений. C ее помощью нельзя, например, проверить принадлежность числа определенному диапазону (для этого используйте конструкцию ветвления - "if").

Задачи:

  1. Написать скрипт, который запрашивает 2 числа, начало и конец диапазона, и в результате выдаёт сумму всех нечетных чисел в этом диапазоне.
  2. Найти факториал числа, у пользователя запрашивается число, и в результате выдаётся факториал числа
  3. Найти последовательность Хэеса, то есть пользователь вводит число и в результате должно быть выдано сообщение, с самой последовательностью, количеством шагов, которое понадобилось для нахождения этой последовательности, и вершиной последовательности, это максимальное число из этой последовательности. Последовательность Хэеса считается так: если ввели число 9, то проверяем его, если оно чётное, делим на 2, если нечетное, умножаем на 3 и прибавляем 1, и так до тех пор пока число не будет равно 1.
  4. Пользователь вводит подряд 20 разных чисел(ввод осуществляется всех чисел только в 1 переменную),нужно посчитать сколько было инверсий.
  5. Пользователь вводит диапазон нужно вывести на экран все простые числа из этого диапазона.
    - Посчитать сумму нечётных элементов массива.
    - Найти сумму элементов массива, элементы которого находятся между крайними отрицательными значениями.
    - Поменять местами элементы массива попарно.
    - Добавить отрицательный элемент массива в случайную позицию в массиве, затем слева от отрицательного элемента , отсортировать элементы по возрастанию, не затрагивая сам отрицательный элемент, а справа по убыванию, так же не затрагивая отрицательный элемент.
    - Посчитать в строке сколько букв, цифр и других символов.
    - Есть массив строк, найти в этом массиве самую короткую строку.
  6. Дано произвольное целое число n. Написать программу, которая
    - разбивает число n на цифры и выводит их на экран;
    - подсчитывает сколько цифр в числе n;
    - находит сумму цифр числа n;
    - меняет порядок цифр числа n на обратный.
  7. Разработать игру "Охота на лис"
    Цель игры состоит в том, чтобы как можно быстрее найти всех лис на игровом поле.
    Игровое поле состоит из массива 10Х10. В нем случайным образом генерится 5 лис (лиса - определяется парой [X,Y]).
    Игрок имеет возможность открыть любую клетку, щелкнув ее левой кнопкой мыши. Если в клетке находится лиса, то она показывается.
    Если лисы нет, в клетке появится цифра, которая указывает, сколько лис находится "по пеленгу" (на горизонтали, вертикали и диагоналях клетки).
    Игра заканчивается после поимки всех лис.
