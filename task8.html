Массивы и методы работы с ними.

Массивы представляют собой набор последовательно проиндексированных элементов Как и многое другое, массив в JavaScript является объектом. Но это с точки зрения архитектуры языка. С нашей же точки зрения его можно представить как объединённую группу переменных, где мы можем работать как с каждой переменной в отдельности, так и со всей группой.

    имя_массива = new Array()

В скобках можно

  • указать количество элементов массива — new Array(8);
  • перечислить элементы массива (в кавычках и через запятую)
    var b = new Array();   // Инициализация переменной.
    b[0] = 'Some text';   // Сохранение значений в массиве.
    b[1] = 10 + 20;
    b[2] = false;
    b[3] = a;   // Сохранение в массиве ещё одного массива.

Обратите внимание, что индексы элементов массива исчисляются от нуля, то есть в массиве «a» длиной в два элемента первый элемент имеет индекс нуль, а второй — один. Другими словами, индекс последнего элемента массива всегда на 1 меньше длины массива.Ещё важно то, что разные элементы массива могут сохранять данные различных типов, например у нас в массиве «a» сохранены число и логическое значение, а в массиве «b» — строка, число, логическое значение и целый массив. Поскольку в ECMAScript типов данных нет, всё перечисленное является возможным.

По умолчанию массив создаётся безмерным, то есть его размер никак не указывается и определяется лишь при заполнении массива.

У массива есть свойство length — длина, или размерность. Это свойство указывает количество элементов массива. У массива с пустыми скобками размерность равна нулю.

Размерность можно динамически изменять. Определив «пустой» массив, можно потом присвоить значение и порядковый номер одному из его элементов. Как только мы это сделаем, изменится и размерность массива: 

    ar = new Array()
    ar[3] = "value"

Обработка массивов

Удобнее всего работать с массивами используя циклы. Узнать длину массива можно с помощью свойства «length». Свойство «length» возвращает длину массива, то есть количество элементов в массиве.

Сортировка массивов

Метод sort сортирует объект Array на месте; во время выполнения новый объект Array не создается.
Если вы предоставляете функцию в аргументе sortfunction, то она должна возвращать одно из следующих значений:

  • Отрицательное значение, если первый прогоняемый аргумент меньше второго аргумента.
  • Ноль, если оба аргумента эквивалентны.
  • Положительное значение, если первый аргумент больше второго аргумента.

Следующий пример показывает использование метода sort: 

    function SortDemo() {
        var a, l;
        a = new Array("X" ,"y" ,"d", "Z", "v","m","r");
        l = a.sort();
        return(l);
    }

Split

Результатом метода split является массив строк, разделенных в точке, где встречается str в stingObj.

Следующий пример показывает использование метода split: 

    function SplitDemo() {
        var s, ss;
        var s = "The quick brown fox jumped over the lazy yellow dog.";
        // Split at each space character.
        ss = s.split(" ");
        return(ss);
    }

slice (Array)

Метод slice возвращает объект Array, содержащий определенную часть arrayObj.

Метод slice копирует, но не включает, элемент, указанный с помощью end. Если аргумент отрицательный, то end указывает на смещение от конца arrayObj. Он не начинается с нуля. Если он опущен, то извлечение продолжается до конца arrayObj.

В следующем примере все, но не последний элемент myArray, копируется в newArray: 

    newArray = myArray.slice(0, -1)

Если ссылка на объект копируется из arrayObj в результат, то ссылка на объект в результате все еще указывает на тот же самый объект. Изменения для такого объекта отражаются в обоих массивах.

slice (String)

Метод slice возвращает объект String, содержащий определенную часть stringObj.

Если аргумент отрицательный, то end указывает на смещение от конца stringObj. Он не начинается с нуля. Если он опущен, то извлечение продолжается до конца stringObj.

В нижеследующем примере оба выражения используют метод slice и возвращают одно и тоже. Отрицательное число во втором выражении указывает на последний символ в str1, как конечную точку: 

    str1.slice(0)
    str2.slice(0,-1)

join

Метод join возвращает объект String, который содержит каждый элемент, конвертированный в строку и соединенный вместе с другими.

Следующий пример показывает использование метода join: 

    function JoinDemo() {
        var a, b;
        a = new Array(0,1,2,3,4);
        b = a.join("-");
        return(b);
    }
